Sources + transparency
Exactly what’s live vs hardcoded
Hardcoded (clearly labeled)
  • Federal tax brackets: IRS Rev. Proc. 2024-40 (2025 brackets). https://www.irs.gov/pub/irs-drop/rp-24-40.pdf
  • SSA wage base: SSA press release (2025). https://www.ssa.gov/news/press/factsheets/colafacts2025.pdf
  • State income tax top rates: NerdWallet table (income earned in 2025). https://www.nerdwallet.com/taxes/learn/state-income-tax-rates
  • State sales tax rates: Sales Tax Institute table. https://www.salestaxinstitute.com/resources/rates
Live (fetched in-browser)
  • Property tax effective rate by state: U.S. Census API (ACS). If blocked, the UI tells you and uses your override.
Model simplifications (explicit)
  • State income tax is modeled via an “effective rate” slider derived from the top marginal rate (so we don’t pretend to implement 50 distinct bracket systems).
  • Retirement uses a 4% rule target portfolio and a deterministic return model (no Monte Carlo in this version).
  • Excise/fees (gas, utilities, telecom, tolls, etc.) are bundled intentionally as a % of spend.
Tip: this is designed to fit on a single 16:9 screen on desktop once sections auto-collapse; on mobile it stays readable and scrolls naturally.
